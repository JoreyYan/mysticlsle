## 🛠️ 本地开发设置

### 前置要求
- Node.js 18+
- npm 或 pnpm

### 快速启动

1. **安装依赖**
   ```bash
   cd frontend
   npm install
   # 或运行 install_deps.bat
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **打开浏览器**
   ```
   http://localhost:3000
   ```

### 环境变量
创建 `frontend/.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=https://vlrhvkislotdojapkbnz.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的anon密钥
NEXT_PUBLIC_APP_NAME=ChillFit Rave
```
